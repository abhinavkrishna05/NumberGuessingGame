# Project Documentation - Number Guessing Game

## 1. Introduction

This document explains the development process, structure, and future plans for the Number Guessing Game project created using Python, GitHub, and Visual Studio Code.

## 2. Objective

The objective of the mini-project is to gain practical experience in:

- Writing and executing code in VS Code
- Using Git for version control
- Hosting and managing a repository on GitHub
- Creating project documentation following industry standards

## 3. Development Process

### Setup
- Created project folder  
- Opened in VS Code  
- Initialized Git repository
- Stage files
- commit changes
- Added remote GitHub URL
- Push changes
